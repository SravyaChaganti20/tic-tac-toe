# 🎮 Tic-Tac-Toe in Python

A console-based Tic-Tac-Toe game developed in Python where a human player competes against the computer. The project demonstrates fundamental Python programming concepts such as functions, loops, conditional statements, lists, and random move generation.

## Features

- Human vs Computer gameplay
- Random computer moves
- Input validation
- Win detection (rows, columns, and diagonals)
- Console-based interface
- Automatic game result display

## Concepts Used

- Functions
- Nested loops
- Conditional statements
- Lists and 2D lists
- Tuples
- User input handling
- Random module
- Game logic

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/python-tic-tac-toe.git
   ```

2. Navigate to the project folder:
   ```bash
   cd python-tic-tac-toe
   ```

3. Run the program:
   ```bash
   python tic_tac_toe.py
   ```
